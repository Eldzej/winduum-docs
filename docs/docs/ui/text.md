# Text

### CSS Properties

```css
:where(.ui-text) {
    --ui-text-size: 0.875rem;
    --ui-text-size-line: 0.75rem;
    --ui-text-weight: var(--font-normal);
    --ui-text-weight-bold: var(--font-semibold);
}
```

* `is-lg` - Large
