# Notice

### CSS Properties

```css
:where(.ui-progress) {
    --ui-progress-height: 1rem;
    --ui-progress-border-radius: var(--rounded-full);
    --ui-progress-bg: var(--color-background-200);
    --ui-progress-bg-opacity: 1;
}
```

### Variants

* `is-sm` - Small size
* `is-lg` - Large size
